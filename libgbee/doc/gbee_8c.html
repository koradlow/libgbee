GBee* gbeeCreate ( const char *  serialName)

Creates a new XBee device. Opens the UART interface the XBee is connected to by calling the GBee port.

Parameters

Returns
A pointer to the GBee device if successful, or NULL in case of any error.

GBeeError gbeeReceive ( GBee *  self,
GBeeFrameData *  frameData,
uint16_t *  length,
uint32_t *  timeout 
)

Read a frame from the XBee and check its validity. This operation calls the serial interface receive operation provided by the port to access the XBee. This operation requires the XBee to be in API mode.

Parameters
[in]selfis a pointer to the XBee device to read from.
[out]frameDatapoints to memory where to store received frame.
[out]lengthis the length of the received frame in bytes.
[in,out]timeoutspecifies a timeout in milliseconds, GBEE_NO_WAIT will cause, the operation to return immediately, GBEE_INFINITE_WAIT sets an infinite timeout. Upon completion of the function, timeout will be set to the remaining time.
Return values
GBEE_NO_ERRORto indicate success.
GBEE_INHERITED_ERRORto indicate that the call failed due to an error in an earlier libgbee call.
GBEE_TIMEOUT_ERRORto indicate that the timeout expired without any data was received.
GBEE_FRAME_INTEGRITY_ERRORto indicate that an incomplete frame was received from the XBee.
GBEE_FRAME_SIZE_ERRORto indicate that the size of the frame received from the XBee exceeds the maximum allowed frame size.
GBEE_CHECKSUM_ERRORto indicate that the checksum failed for the received frame.

GBeeError gbeeXferAtCommand ( GBee *  self,
const char *  command,
const char *  args,
uint16_t  argLength,
char *  response,
uint16_t *  responseLength 
)

Transfers the given AT command to the XBee and provides the result. This operation requires the XBee to be in transparent mode. Note, that the transmission of the AT command in transparent mode may take approximately 2-3 seconds!

Parameters
[in]selfis a pointer to the XBee device.
[in]commandis the command.
[in]argsare the command arguments as 0-terminated string.
[in]argLengthis the length of arguments.
[out]responseis the XBee response (up to GBEE_MAX_FRAME_SIZE bytes).
[out]responseLengthis the length of the XBee response.
Returns
GBEE_NO_ERROR if successful, or dedicated error code in case of any error.
Return values
GBEE_RS232_ERRORto indicate a failure to establish serial communication with the XBee.
GBEE_RESPONSE_ERRORto indicate that the XBee replied with an error code.
